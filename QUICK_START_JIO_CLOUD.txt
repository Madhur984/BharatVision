## Deploy to Jio Cloud (Quick Steps)

### Option 1: Docker (Easiest)

1. Build image:
   ```bash
   docker build -f Dockerfile.cloud -t bharatvision-ml-api .
   ```

2. Test locally:
   ```bash
   docker run -p 8000:8000 --env-file .env.cloud bharatvision-ml-api
   ```

3. Push to Jio Cloud and deploy

### Option 2: Direct VM

1. Upload files to Jio Cloud VM
2. Install dependencies: `pip install -r requirements_api.txt`
3. Run: `python simple_api.py`

## Configure Streamlit Cloud

1. Go to https://share.streamlit.io/
2. Settings → Secrets
3. Add:
   ```toml
   ML_API_URL = "https://your-jio-cloud-api-url"
   ```
4. Save and reboot

## Test
