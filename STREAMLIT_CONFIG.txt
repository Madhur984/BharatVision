# Streamlit Cloud Configuration

## Add this to your Streamlit Cloud Secrets

1. Go to: https://share.streamlit.io/
2. Find your `bharatvision` app
3. Click **Settings** (⚙️) → **Secrets**
4. Add this line:

```toml
ML_API_URL = "https://madhur984-bharatvision-ml-api.hf.space"
```

5. Click **Save**
6. Click **Reboot app**
